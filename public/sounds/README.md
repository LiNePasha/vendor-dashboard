# Notification Sound

## كيفية إضافة صوت التنبيه:

### الطريقة 1: استخدام صوت من الإنترنت (مؤقت)
الكود الحالي يبحث عن ملفات:
- `notification.mp3`
- `notification.ogg`

### الطريقة 2: تحميل صوت جاهز
يمكنك تحميل صوت تنبيه مجاني من:
1. https://notificationsounds.com/
2. https://mixkit.co/free-sound-effects/notification/
3. https://freesound.org/

### الطريقة 3: إنشاء صوت باستخدام JavaScript
يمكن استخدام Web Audio API لإنشاء صوت بسيط.

### مؤقتاً:
الكود سيعمل بدون صوت، وعند إضافة الملف سيبدأ في التشغيل تلقائياً.

---

## بديل سريع - استخدام Data URI:

يمكنك استخدام صوت beep مباشرة من الكود بدون ملف خارجي.
